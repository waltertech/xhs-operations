# 小红书运营 Skill

## 描述
当你需要运营小红书账号时使用此skill。支持全流程运营或分步执行。

## 可用功能

### 完整流程
- 一键运营：自动完成选题→筛选→撰写→配图→发布

### 分步功能
- 选题发现：从微信公众号/X/小红书/Reddit搜索热门内容
- 选题筛选：对选题进行综合评分排序
- 内容撰写：生成小红书风格种草文案
- 标题生成：生成高CTR优化标题
- 配图设计：HTML截图或AI生图
- 自动发布：发布笔记到小红书

## 参数

- topic: 运营主题/关键词
- mode: 执行模式 (full/discovery/filter/write/title/image/publish)
- platforms: 搜索平台列表 (wechat/x/xhs/reddit)

## 示例

用户: "帮我运营小红书，主题是AI工具推荐"
-> 执行完整流程

用户: "帮我找一些AI领域的选题"
-> 执行选题发现 + 筛选
