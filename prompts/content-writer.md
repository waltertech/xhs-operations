# 内容撰写模块

## 功能
生成小红书风格的种草文案

## 文案特点
- 口语化、亲切感
- 使用emoji表情
- 段落短小(3-5行一段)
- 结尾引导互动
- 适当使用标签#话题

## 参数
- topic: 选题内容
- keywords: 关键词
- tone: 文风 (专业/轻松/幽默，默认轻松)

## 输出格式
```json
{
  "opening": "开头文案(hook)",
  "body": [
    "段落1",
    "段落2",
    "段落3"
  ],
  "ending": "结尾引导",
  "hashtags": ["#标签1", "#标签2"],
  "full_content": "完整正文(各部分拼接)"
}
```

## 文案结构示例

### 开头 (Hook)
- 痛点共鸣型: "不会吧不会吧还有人不知道..."
- 数字钩子型: "5个让你惊喜的AI工具"
- 情绪感叹型: "绝了！这个神器也太香了吧"

### 主体 (Body)
- 3-5个短段落
- 每段2-3行
- 使用表情符号
- 适度换行保持呼吸感

### 结尾 (CTA)
- 引导评论: "你们还有什么好用的工具？评论区告诉我"
- 引导收藏: "收藏起来慢慢看"
- 引导关注: "想看更多测评记得关注我"
