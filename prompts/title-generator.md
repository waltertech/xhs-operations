# 标题生成模块

## 功能
生成高CTR的小红书标题

## 标题策略

### 数字钩子
- "5个技巧"、"3步搞定"、"10款必备"
- 示例: "5个让你效率翻倍的AI工具"

### 情绪钩子
- "必看！"、"绝了"、"太香了"、"惊艳"
- 示例: "绝了这个AI写作神器！"

### 悬念钩子
- "原来..."、"居然..."、"99%的人不知道"
- 示例: "原来AI还能这样用"

### 痛点共鸣
- "不会还有人不知道..."、"拯救你的..."
- 示例: "不会还有人不知道这款神器吧"

### 对比冲突
- "VS"、"对比"、"完胜"
- 示例: "ChatGPT VS Claude 到底谁更强"

## 参数
- content: 笔记正文
- style: 标题风格(可选)
- count: 生成数量(默认5个)

## 输出格式
```json
{
  "titles": [
    {
      "title": "生成的标题",
      "strategy": "使用的策略",
      "ctr_score": 0.85
    }
  ]
}
```

## CTR评分标准
- 长度: 18-25字为最佳 (0.2)
- 数字: 包含数字+0.1
- 情绪词: 包含情绪词+0.1
- 悬念: 制造悬念感+0.15
- 痛点: 引发共鸣+0.15
- 独特词汇: 有记忆点+0.1
